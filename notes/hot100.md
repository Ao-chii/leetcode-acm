# Hot 100 Python 实用速查

## 输入输出

LeetCode 通常不用自己处理输入输出，但笔试、ACM 模式会用到。

| 场景                    | 写法                                                    | 说明         |
| ----------------------- | ------------------------------------------------------- | ------------ |
| 读一行字符串            | `s = input().strip()`                                 | 去掉首尾空白 |
| 读一行整数              | `x = int(input())`                                    | 单个数字     |
| 读一行多个整数          | `nums = list(map(int, input().split()))`              | 最常用       |
| 读固定 `n` 行         | `[input().strip() for _ in range(n)]`                 | 字符串列表   |
| 读固定 `n` 行整数数组 | `[list(map(int, input().split())) for _ in range(n)]` | 矩阵         |
| 快速读入                | `sys.stdin.readline().strip()`                        | 数据大时使用 |

```python
import sys

n = int(sys.stdin.readline())
nums = list(map(int, sys.stdin.readline().split()))
```

## 字符串常用操作

| 方法              | 返回类型      | 是否修改原字符串 | 主要用途             |
| ----------------- | ------------- | ---------------- | -------------------- |
| `split()`       | `list[str]` | 否               | 按空白切割字符串     |
| `split(sep)`    | `list[str]` | 否               | 按指定分隔符切割     |
| `strip()`       | `str`       | 否               | 删除首尾空白         |
| `replace(a, b)` | `str`       | 否               | 替换子串             |
| `join(list)`    | `str`       | 否               | 拼接字符串列表       |
| `startswith(x)` | `bool`      | 否               | 判断前缀             |
| `endswith(x)`   | `bool`      | 否               | 判断后缀             |
| `isdigit()`     | `bool`      | 否               | 判断是否全是数字字符 |

```python
s = "  1 2 3  "
nums = list(map(int, s.strip().split()))
ans = " ".join(map(str, nums))
```

## Python 容器

| 容器            | 本质           | 高频用途                 | 关键操作                         |
| --------------- | -------------- | ------------------------ | -------------------------------- |
| `list`        | 动态数组       | 双指针、栈、排序、前缀和 | `append`、`pop`、`sort`    |
| `dict`        | 哈希表         | 映射、计数、两数之和     | `get`、`in`、`items`       |
| `set`         | 哈希集合       | 去重、判存在、交并差     | `add`、`remove`、`discard` |
| `tuple`       | 不可变数组     | 坐标、哈希 key           | `(x, y)`                       |
| `deque`       | 双端队列       | BFS、滑动窗口、单调队列  | `append`、`popleft`          |
| `defaultdict` | 带默认值的字典 | 分组、建图、计数         | `defaultdict(list)`            |
| `Counter`     | 计数器         | 字符频次、众数、窗口计数 | `most_common`                  |
| `heapq`       | 小根堆         | Top K、贪心、优先队列    | `heappush`、`heappop`        |

### `list`

```python
nums.sort()
nums.sort(reverse=True)

stack = []
stack.append(x)
top = stack.pop()
```

注意：`list.pop(0)` 是 O(n)，队列场景用 `deque.popleft()`。

### `dict`

```python
cnt = {}
for x in nums:
    cnt[x] = cnt.get(x, 0) + 1

if target in cnt:
    pass
```

别反复写“先判断 key 在不在”。代码越啰嗦，分支越容易错。

### `set`

```python
seen = set()
for x in nums:
    if x in seen:
        return True
    seen.add(x)
```

`remove(x)` 在元素不存在时报错，`discard(x)` 不会报错。

### `defaultdict`

graph = defaultdict(list)

### `Counter`

```python
from collections import Counter

cnt = Counter(s)
most = cnt.most_common(1)
```

窗口题里可以直接比较两个 `Counter`，但频繁比较会有开销。性能卡住时，用变量维护“满足条件的字符数”。

### `deque`

```python
from collections import deque

q = deque([root])
while q:
    node = q.popleft()
```

单调队列模板：

```python
from collections import deque

q = deque()
for i, x in enumerate(nums):
    while q and nums[q[-1]] <= x:
        q.pop()
    q.append(i)
    while q and q[0] <= i - k:
        q.popleft()
    if i >= k - 1:
        ans.append(nums[q[0]])
```

### `heapq`

Python 只有小根堆。需要大根堆时存负数。

```python
import heapq

heap = []
for x in nums:
    heapq.heappush(heap, x)

smallest = heapq.heappop(heap)
```

Top K 小根堆：

```python
import heapq

heap = []
for x in nums:
    heapq.heappush(heap, x)
    if len(heap) > k:
        heapq.heappop(heap)
```

## 复杂度速记

| 操作                             | 平均复杂度 | 备注       |
| -------------------------------- | ---------- | ---------- |
| `x in list`                    | O(n)       | 线性查找   |
| `x in set` / `x in dict`     | O(1)       | 哈希查询   |
| `list.append` / `list.pop()` | O(1)       | 尾部操作   |
| `list.pop(0)`                  | O(n)       | 会整体搬移 |
| `deque.popleft()`              | O(1)       | 队列首选   |
| `sort()`                       | O(n log n) | 原地排序   |
| `heapq.heappush/pop`           | O(log n)   | 优先队列   |
